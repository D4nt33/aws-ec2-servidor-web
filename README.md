<img width="921" height="589" alt="image" src="https://github.com/user-attachments/assets/cfb63344-aecb-4120-9ee9-41e7cb33c7b1" /># aws-ec2-servidor-web
Este proyecto muestra cómo desplegar un servidor web en AWS usando EC2 y Nginx.
✔️ Tecnologías:
AWS EC2
Ubuntu
Nginx
✔️ Pasos (importante):
Crear instancia
Conectarse por SSH
Instalar Nginx
✔️ Problema real:
Al reiniciar la instancia, la IP pública cambió debido a que AWS asigna IPs dinámicas. 
Solución: usar Elastic IP (pendiente de implementar).
